# block-world-JS

### Usage
  - clone the project
  - run npm install
  - then run npm start
  - use the cmd to enter input required 

### Description
  - NB: assumes that a is always smaller than b (a is in a position below b)

  #### moves
    - move a onto b
    - move a over b
    - pile a onto b
    - pile a over b
    - quit
